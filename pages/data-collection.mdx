## Disclaimer
This website **NEVER** collect any your personal information. This website doesn't use any analytic tools (e.g Google Analytics, etc).

## Privacy Concern
Since this website is hosted on [Netlify](https://www.netlify.com), this website follows [Netlify Privacy Policy](https://www.netlify.com/privacy/). If you have any privacy concerns about this website, please contact [Netlify Customer Service](https://www.netlify.com/contact/) directly, because I do not have such power to collect any your personal/sensitive information.
This website use proxy service. I used **Cloudflare** for the proxy service.

## Web Tech Stack
- Runtime Environment: [Node.js](https://nodejs.org)
- Framework: [Next.js](https://nextjs.org) & [Nextra](https://nextra.site)
- PaaS (Platform as a Service): [Netlify](https://www.netlify.com)
- Proxy Service: [Cloudflare](https://www.cloudflare.com)
